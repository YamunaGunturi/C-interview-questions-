1. What is C?
Answer:
C is a general-purpose, procedural programming language developed by Dennis Ritchie in 1972. It is widely used for system programming, embedded systems, and application development

2. What are the features of C?
Answer:
Simple and easy to learn
Fast execution
Portable
Structured programming
Supports pointers
Rich standard library

3. Why is C called a middle-level language?
Answer:
C is called a middle-level language because it supports both high-level programming features (functions, portability) and low-level features (pointers, direct memory access).

4. What are keywords in C?
Answer:

Keywords are reserved words that have predefined meanings and cannot be used as variable names.
Examples: int, if, while, return, for.

5. What is a variable?
   
Answer:
A variable is a named memory location used to store data.
Example:
int age = 20;

6. What are the basic data types in C?
Answer:

int – Integer values
char – Single character
float – Decimal values
double – Double-precision decimal values
void – No value


7. What is the difference between printf() and scanf()?
Answer:

printf() is used to display output.
scanf() is used to take input from the user.


8. What is the difference between = and ==?
Answer:

= assigns a value.
== compares two values.

9. What are operators in C?
Answer:

Operators are symbols used to perform operations on variables.
Types:

Arithmetic (+, -, *, /, %)
Relational (>, <, ==)
Logical (&&, ||, !)
Assignment (=, +=)
Bitwise (&, |, ^)

10. What are loops in C?
Answer

Loops are used to execute a block of code repeatedly.
Types:

for
while
do-while
